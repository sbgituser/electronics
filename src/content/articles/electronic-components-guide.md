## 電子部品の基礎を学ぶ重要性

電子工作を始めると、抵抗・コンデンサ・トランジスタなど様々な部品が登場します。これらの部品は回路図では記号で表され、それぞれが決まった役割を持っています。部品の役割を理解することで、回路図が読めるようになり、自分で回路を設計する力がつきます。

この記事では電子工作でよく使う基本部品を5種類解説します。部品の購入には秋月電子通商（akizukidenshi.com）や千石電商が品揃え豊富でおすすめです。

## 抵抗（レジスタ）

### 役割と原理

抵抗は電流の流れを制限する部品です。単位はオーム（Ω）で、値が大きいほど電流が流れにくくなります。オームの法則（V = I × R）に従い、電圧・電流・抵抗の関係を計算できます。

電子工作での主な使い方は：
- **LEDの保護抵抗**：LED電流を適切な値に制限する
- **プルアップ・プルダウン抵抗**：デジタル入力ピンを安定したHigh/Low状態に保つ
- **電圧分圧**：2つの抵抗でアナログセンサー値を読み取りやすい電圧に変換する

### カラーコードの読み方

カーボン抵抗には値を示す色帯（カラーコード）が印刷されています。

| 色 | 数値 | 乗数 |
|----|------|------|
| 黒 | 0 | ×1 |
| 茶 | 1 | ×10 |
| 赤 | 2 | ×100 |
| 橙 | 3 | ×1k |
| 黄 | 4 | ×10k |
| 緑 | 5 | ×100k |
| 青 | 6 | ×1M |
| 紫 | 7 | - |
| 灰 | 8 | - |
| 白 | 9 | - |
| 金 | - | ×0.1（誤差±5%） |
| 銀 | - | ×0.01（誤差±10%） |

4本帯の読み方：[1桁目][2桁目]×[乗数]（誤差）

**例：赤赤橙金** → 2・2・×1000・±5% → **22kΩ（±5%）**

慣れないうちは抵抗値を実測する方が確実です。テスターの抵抗レンジで測ると正確な値が確認できます。

### 定格電力

抵抗には定格電力があり、これを超えて使うと焦げます。一般的な1/4W（0.25W）抵抗は、抵抗値×電流の二乗が0.25W以下の用途で使えます。大きな電流が流れる箇所では1/2W、1W、2W などの大電力型を選びます。

## コンデンサ（キャパシタ）

### 役割と原理

コンデンサは電荷を蓄えることができる部品です。単位はファラド（F）ですが、一般的な電子工作では極めて小さな値（ナノファラド nF、マイクロファラド μF）が使われます。

主な用途は：
- **デカップリング（バイパス）**：電源ラインのノイズ除去。マイコンのVCC-GND間に100nFのセラミックコンデンサを入れるのが定石。
- **タイミング回路**：抵抗とコンデンサを組み合わせたCR回路で時定数を作る。
- **平滑（フィルタリング）**：交流成分を除去して直流を安定させる。

### コンデンサの種類

| 種類 | 特徴 | 極性 | 主な用途 |
|------|------|------|---------|
| セラミックコンデンサ | 小型、安価 | なし | デカップリング、高周波フィルタ |
| 電解コンデンサ | 大容量、極性あり | あり | 電源平滑、大容量バイパス |
| タンタルコンデンサ | 安定、小型、高価 | あり | 精密な回路 |
| フィルムコンデンサ | 高精度、低損失 | なし | オーディオ、タイミング回路 |

**注意：電解コンデンサは極性があります。**リード線の長い方がプラス（アノード）、短い方がマイナスです。また本体に白いストライプが入っている側がマイナスです。逆接続すると破損・爆発の危険があるため、必ず確認してから取り付けましょう。

## ダイオード

### 役割と原理

ダイオードは電流を一方向にしか通さない半導体部品です。アノード（+）からカソード（-）へは電流が流れますが、逆方向はブロックします。

電子工作での主な使い方：
- **整流**：交流を直流に変換（電源回路）
- **逆電圧保護**：電源の接続を間違えてもマイコンを守る保護回路
- **フライバックダイオード**：モーターやリレーが発生する逆起電力をバイパス

一般的な整流ダイオード1N4007は順方向電圧が約0.7Vで、1Aまでの電流に対応しています。LEDもダイオードの一種で、光を発する機能が追加されています。

## トランジスタ

### 役割と原理

トランジスタは「電子的なスイッチ」または「増幅器」として機能する半導体部品です。NPN型バイポーラトランジスタを例に説明します。

**ピン構成（TO-92パッケージの例）：**
- **B（ベース）**：制御入力。わずかな電流を流すことで
- **C（コレクタ）**：大きな電流の入り口
- **E（エミッタ）**：電流の出口、通常GNDに接続

ベースに制御電流（IB）を流すとコレクタ-エミッタ間が導通します。電流増幅率（hFE）が100なら、ベース電流の100倍のコレクタ電流が流せます。

### ArduinoでLEDや小型モーターをトランジスタで制御する

```cpp
const int BASE_PIN = 9;  // トランジスタのベースに接続

void setup() {
  pinMode(BASE_PIN, OUTPUT);
}

void loop() {
  digitalWrite(BASE_PIN, HIGH);  // トランジスタON（モーター回転）
  delay(2000);
  digitalWrite(BASE_PIN, LOW);   // トランジスタOFF（停止）
  delay(2000);
}
```

**配線のポイント：**
- ArduinoのデジタルピンとトランジスタのB（ベース）の間に1kΩ抵抗を入れる
- 負荷（LED・モーター）はコレクタとVCCの間に接続
- エミッタはGNDに接続
- モーターにはフライバックダイオード（1N4007）を並列に接続する

入門用のNPNトランジスタとしては**2SC1815**（日本製定番品、hFE=70〜700）や**BC548**（海外定番品）が汎用的に使えます。

## オペアンプ（OP-Amp）

オペアンプは差動増幅器で、2つの入力端子の電位差を増幅します。入門段階では深く学ばなくて良いですが、センサー信号の増幅や比較回路に使われることを覚えておきましょう。LM358やLM741が入門向けの汎用品です。

## 集積回路（IC）

ICは複数のトランジスタ・抵抗・コンデンサを一つのパッケージに収めた部品です。555タイマーICは発振回路・タイマー回路に広く使われる入門定番ICで、¥50〜100と非常に安価です。ArdinoなどのマイコンもすべてICの一種です。

## 部品の保管と管理のコツ

電子部品は静電気・湿気・熱に弱いものがあります。

| 部品 | 注意点 | 保管方法 |
|------|--------|---------|
| CMOS IC | 静電気で破壊 | 導電性スポンジ・袋に保管 |
| 電解コンデンサ | 液漏れ注意 | 常温・乾燥した場所に保管 |
| 抵抗・LED | 比較的強い | 仕切りケースに種類別で保管 |

部品の整理には仕切りの多い透明なプラスチックケースが便利です。抵抗はカラーコード別または値別に分けて保管すると、必要な時にすぐ取り出せます。

## まとめ

電子部品の基礎をまとめます。

- **抵抗**：電流制限・分圧。カラーコードで値を読む。
- **コンデンサ**：電荷の蓄積。電解は極性に注意。デカップリングに必須。
- **ダイオード**：一方向のみ電流を通す。保護回路に活用。
- **トランジスタ**：電子スイッチ。大電流負荷をArduinoピンで制御するのに使う。
- **IC**：複数の回路を1パッケージに集積。555タイマーは入門定番。

これらの部品を使いこなすことが電子回路設計の第一歩です。実際に[Arduino入門ガイド](/articles/arduino-beginner-guide)を参照しながら回路を組んでみましょう。

## よくある質問

**Q. 抵抗のカラーコードはどうやって読みますか？**

4本帯の場合、左から1桁目、2桁目、乗数、誤差を表します。例えば赤赤橙金は22×1000=22kΩ（誤差±5%）です。オームメーターで実測するのが確実で簡単です。

**Q. コンデンサの極性（プラスマイナス）を間違えると壊れますか？**

電解コンデンサは極性があり、逆接続すると破損・爆発の危険があります。リード線の長い方がプラス（アノード）です。セラミックコンデンサには極性はありません。

**Q. トランジスタはArduinoでどう使いますか？**

Arduinoのピンから直接動かせない大電流負荷（モーター等）を制御するスイッチとして使います。ベース電流を流すとコレクタ-エミッタ間が導通します。2SC1815などの汎用NPNトランジスタが入門に向いています。

**Q. 電子部品はどこで購入できますか？**

秋月電子通商、千石電商、マルツオンライン、Amazonなどで購入できます。初心者はセット品を購入すると割安で種類も揃います。
